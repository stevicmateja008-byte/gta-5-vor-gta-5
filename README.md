# Installation Instructions for Running the GTA-5-like Game

## Python Version Requirements

Make sure you have Python 3.7 or higher installed on your machine.

## Installing Dependencies

To install the required dependencies, run the following command:
```bash
pip install -r requirements.txt
```

## Starting the Game

Once you have installed the dependencies, you can start the game by running:
```bash
python main.py
```

## Controls Documentation

### Keyboard Controls
- **W**: Move Forward
- **S**: Move Backward
- **A**: Move Left
- **D**: Move Right
- **Space**: Jump
- **Left Shift**: Sprint
- **E**: Interact/Use Item
- **Escape**: Open Menu

### Mouse Controls
- **Left Click**: Action/Select
- **Right Click**: Secondary Action
- **Scroll Wheel**: Zoom In/Out

Enjoy your gaming experience!
